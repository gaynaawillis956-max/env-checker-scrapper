================================================
TROUBLESHOOTING
================================================

If RUN-DESKTOP.bat doesn't work:
→ Use RUN-WEB.bat instead (more reliable)
→ Or open command prompt and run: python3 start.py web

If RUN-WEB.bat doesn't open browser:
→ Manually open: http://127.0.0.1:5000

If port 5000 is in use:
→ Edit RUN-WEB.bat
→ Change: python3 start.py web
→ To:      python3 start.py web --port 8080

If Python is not found:
→ Install Python 3.8+ from https://www.python.org/
→ Make sure "Add Python to PATH" is checked
