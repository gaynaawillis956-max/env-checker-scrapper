 📧 MASS MAILER WORKFLOW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Step 1: Start Web Server
    → RUN-WEB.bat
    → Open http://127.0.0.1:5000

  Step 2: Upload SMTP Accounts
    → Go to "Mass Mailer" tab
    → Upload smtp_list.txt (format: host:port:user:pass)
    → OR use saved credentials from MANAGE-PASSWORDS.bat

  Step 3: Enter Recipients
    → Paste email list or upload file
    → Format: one email per line

  Step 4: Select Template or Write HTML
    → Choose from "Use Template" dropdown
    → OR paste custom HTML in "Email Body"
    → Select subject line

  Step 5: Send
    → Click "Send"
    → Monitor progress in real-time
    → View success/fail counts
